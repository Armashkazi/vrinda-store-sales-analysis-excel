# Vrinda Store Sales Analysis (2022)

## Project Overview

This project analyzes retail sales data of Vrinda Store for the year 2022 to understand customer behavior, sales performance, and business trends. The goal was to clean the dataset, perform analysis, and build an interactive dashboard to support business decision-making.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Charts & Dashboard
* Data Cleaning Techniques

## Data Preparation

* Removed duplicates and null values
* Standardized columns and corrected data types
* Created calculated columns for analysis
* Organized dataset for reporting

## Analysis Performed

* Monthly sales trend analysis
* Gender-wise sales analysis
* Age group segmentation
* State-wise revenue analysis
* Order channel performance analysis

## Key Insights

* Female customers contributed the majority of total sales.
* Customers aged 30–49 years formed the largest purchasing segment.
* Maharashtra, Karnataka and Uttar Pradesh generated the highest revenue.
* Amazon, Flipkart and Myntra were the top-performing sales channels.

## Dashboard Features

* Interactive filters (Month, Category, Channel)
* KPI indicators (Total Sales, Orders, Revenue)
* Visual charts for trend comparison
* Easy-to-understand business reporting

## Conclusion

The dashboard helps stakeholders quickly understand sales performance and customer behavior, enabling better marketing strategies and business planning.
